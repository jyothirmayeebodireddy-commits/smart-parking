# Vel Tech Smart Parking System

AI-powered smart parking management system built 
for Vel Tech University.

## Team
- Chitra — AI / ML Engineer
- Jyothi — Backend Developer  
- Chetan — Frontend Developer (that's me!)

## My Part (Frontend)
Built with React, Recharts, Axios, React Router DOM

### Pages I Built
- Dashboard — Live parking grid with auto-refresh
- Reserve — Slot booking with QR token
- Predictions — AI forecast charts
- Login & Register — User authentication
- 404 — Custom error page

## How to Run Frontend
1. cd smart-parking-frontend
2. npm install
3. npm start
4. Open http://localhost:3000

## Tech Stack
- React
- React Router DOM
- Recharts (charts)
- Axios (API calls)
- Custom CSS animations
